Cryptography Service Provider (CSP), криптопровайдер

CSP - посредник, кот.позводяет ОС выполнять шифровальные функции
CSP - это спец.программа для работы с ЭЦП, обеспечивающая соблюдение
      стандартов (ГОСТ)
CSP - это предоставляющая специальный API и спец.образом зарегистрированная
      в ОС библиотека, кот.позволяет расширить список поддерживаемых
      в ОС криптоалгоритмов

Объектры криптопровайдера
Основным объектом криптопровайдера выступает ключевой контейнер.
Контейнер имеет собственное имя. Каждый ключевой контейнер может содержать:
  - максимум 1 пару ключей ЭП
  - максимум 1 пару ключей обмена
  - максимум 1 симметричный ключ

Вне контейнера может находиться исключительно открытый ключ.
Для вычисления хэш-функций используются спец.объекты хеширования. Для их создания
наличие контейнера не требуется

Популярные в РФ криптопровайдеры:
  - КриптоПро CSP
  - ViPNet CSP (бесплат., серт ФСБ, Win, Linux)
  - Signal-COM CSP (только под Win)

Для того, чтобы из скриптов WEB-страницы вызвать нативную библиотеку, большинство
браузеров поддерживают спец.расширения (Плагины?)
